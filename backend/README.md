# Aurik 4.0 Backend

Dieses Verzeichnis enthält die Kernmodule für das SOTA-Backend:

- file_import.py: Laden und Validieren von Audiodateien
- feature_analysis.py: Feature-Extraktion, Policy-Analyse
- processing.py: DSP-/ML-Module, Bearbeitungslogik
- exporter.py: Exportfunktionen

Alle Module sind modular, testbar und für die Integration mit dem Frontend vorbereitet.
